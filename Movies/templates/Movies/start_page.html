Welcome!

This site has been designed as a part of my final year (hopefully) as a computer science student. The aim of this project is to analyse three recommender systems using different types of machine learning algorithm.
There are 3 steps to complete :

Step 1 :
Here on this page we ask you for some information about yourself. This information will be used to determine if age or how often a person watches movies has any effect on which recommender is more suited to them.

Step 2 :
You will be presented with a list of movies, we ask that you rate any of the movies you have seen in this list so we can get a sense of the types of movies you enjoy.

Step 3 :
We will present you with recommendations from each of the algorithms. We will show you the first round of recommendations from each algorithm on separate pages and ask you to rate each movie and let us know if you have watched it. Then we will repeat this cycle for a second round. (so we can see how much the algorithms learn). 3 x pages of recommendation for 2 rounds. 6 pages in total.

Step 4 :
We thank you profusely and present to you a lovely graph that shows you which genres of movie you tend to favour
